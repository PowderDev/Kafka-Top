### What it does?
- Python service grabs cpu usage data and produces it to Kafka
- Go service consumes the message and broadcast it to all websockets connections
- React app shows a dynamic chart

### What are the tools?
- Kubernetes
- Kafka
- Python
- Go
- TypeScript
